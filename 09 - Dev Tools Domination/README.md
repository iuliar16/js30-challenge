# PROJECT DESCRIPTION
This project showcases some dev tools methods, such as:

console.log - used to log something in dev tools

use %c in log with some css to style the message

console.warn to show the message as a warning

console.error to show it as an error

console.info to show it as an information

add html elements in console.log to see the element and to see more info about it you can add it in console.dir

console.clear to clear the console

console.count to count the number of times an element was mentiones

console.time and console.timeEnd to measure time
console.table to show info as a table
